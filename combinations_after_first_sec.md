Initial Extra Deck:
- Xyz Ranks: 2, 3, 4, 5
- Fusion Levels: 1, 2, 3, 4, 5, 6

use gen_sec.py to generate for other Extra Deck combinations

*Assumption: You run exactly 1 copy of each in your Extra Deck, so the cards banished during the first SEC are no longer available to send for the second SEC.*

### Legend:
- 🌟 : **Freebie Level** (Opponent monster equals $X_1 + F_1$. Any valid $T$ resolves the card because you just return your previously banished monsters!)
- ✅ : **Valid State** (Standard successful resolution)
- 🆕 : **Newly Possible State** (This combination was impossible at the start, but is now possible because you can mix-and-match newly banished cards with previously banished ones!).
- ❌ : **Lost State** (This $(L, T)$ combination was solvable at the start of the duel, but became impossible because you banished the necessary cards).
- (Empty) : Impossible state (Was impossible at the start, and is still impossible).
